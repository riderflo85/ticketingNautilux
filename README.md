# Ticket Nautilux
## Interface de suivie d'intervention par tiquet.
### Projet réalisé le 30/09/2021 par GRENAILLE Florent.

# Dépendances:
- pipenv
- Python 3.8
- Django 3.2.7
- AngularJS 1.7.9

# Installation
Installer [pipenv](https://pypi.org/project/pipenv/) via pip
```shell
$ pip install pipenv
```

Installer le projet et ces dépendances.
```shell
$ pipenv install
```
